Installing Cruftman
===================

Use composer_ to install the cruftman project.

.. code-block:: shell

      composer create-project "cruftman/cruftman:dev-master"

.. note::

   We have not released Cruftman yet, so it's only installable from
   github_.

.. _composer: https://getcomposer.org/
.. _github: https://github.com/cruftman/cruftman/
